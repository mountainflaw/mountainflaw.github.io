modconv

modconv is a model converter specifically designed for importing custom assets into the Super Mario 64 decompilation.

How to use

Command line syntax


./modconv [PATH TO MODEL] [OUTPUT DIRECTORY] [SCALE] [OUTPUT TYPE]

Capabilities

modconv can output to F3D, F3DEX, F3DEX Rej 64 vertex cache, F3DEX Rej 80 vertex cache, and collision. Goddard (Mario head) importing will be available soon.
