◆ gd_get_colour()

struct GdColour* gd_get_colour ( s32  idx)

Convert a numeric index into pointer to a struct GdColour.

A simple switch case to convert from index idx to a pointer to the three float GdColour structure. Goddard stored the index in a structure, and uses this function to get the colour RGB values if needed. -1 uses the environment colour. A possible enhancement for this is to ennumerate all colours, and then use those enumerations and/or enum type where ever a colour is requested

Parameters
idxIndex of colour
Returns
Pointer to a GdColour struct
